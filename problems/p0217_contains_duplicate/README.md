# Problem 0217: Contains Duplicate

Link: https://leetcode.com/problems/contains-duplicate/

## Statement

Given integer array `nums`, return `true` if any value appears at least twice, `false` if every element distinct.

## Constraints

- `1 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`

## Examples

- Input: `nums = [1,2,3,1]`
  Output: `true`
- Input: `nums = [1,2,3,4]`
  Output: `false`
- Input: `nums = [1,1,1,3,3,4,3,2,4,2]`
  Output: `true`

## Target

- Time: O(n)
- Space: O(n)
